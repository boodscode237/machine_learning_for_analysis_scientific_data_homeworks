Прогнозирование цены продажи бульдозеров с помощью машинного обучения.¶

В этой тетради мы рассмотрим пример учебного проекта, цель которого - предсказать цену продажи бульдозеров.

1. Определение проблемы¶

Как хорошо мы можем предсказать будущую цену продажи бульдозера, учитывая его характеристики и предыдущие примеры того, за сколько были проданы аналогичные бульдозеры?".

2. Данные.¶

Данные загружены с конкурса Kaggle Bluebook for bulldozers:

  • Train.csv - это обучающий набор, содержащий данные до конца 2011 года.
  • Valid.csv - это набор для проверки, который содержит данные с 1 января 2012 года по 30 апреля 2012 года Вы делаете прогнозы на этом наборе на протяжении большей части соревнования. Ваш результат по этому набору используется для создания публичной таблицы лидеров.
  • Test.csv - это тестовый набор, который не будет опубликован до последней недели конкурса. Он содержит данные с 1 мая 2012 года по ноябрь 2012 года. Ваш результат в тестовом наборе определяет ваше итоговое место в конкурсе.

Для получения дополнительной оценки этого проекта проверьте конкурс.

3. Оценка.¶

Внимание, что целью большинства показателей оценки регрессии является минимизация ошибки. Например, целью этого проекта является создание модели машинного обучения, которая сводит к минимуму RMSLE.

from sklearn.metrics import mean_squared_log_error, mean_absolute_error, r2_score

def rmsle(y_test, y_preds):
  """
  Calculates root mean squared log error between predictions and true labels.
  """

  return np.sqrt(mean_squared_log_error(y_test, y_preds))

# Create function to evaluate model on few different levels

def show_scores(model):
  train_preds = model.predict(X_train)
  val_preds = model.predict(X_valid)
  scores = {
      "Training MAE": mean_absolute_error(y_train, train_preds),
      "valid MAE": mean_absolute_error(y_valid, val_preds),
      "Training RMSLE": rmsle(y_train, train_preds),
      "Valid RMSLE": rmsle(y_valid, val_preds),
      "Training R^2": r2_score(y_train, train_preds),
      "valid R^2": r2_score(y_valid, val_preds),
  }

  return scores

import matplotlib.pyplot as plt
import numpy as np

def plot_feature_importance(columns, importances, n=20):
    """
    Plot feature importance based on importances and feature names.

    Parameters:
    - columns: Feature names (list or array-like)
    - importances: Feature importances (list or array-like)
    - n: Number of top features to display in the plot (default: 20)
    """
    df = (pd.DataFrame({"features": columns,
                        "feature_importances": importances})
    .sort_values("feature_importances", ascending=False)
    .reset_index(drop=True))

    # Plot the DataFrame
    fig, ax = plt.subplots()
    ax.barh(df["features"][:n], df["feature_importances"][:n])
    ax.set_ylabel("Features")
    ax.set_xlabel("Feature Importance")
    ax.invert_yaxis()
